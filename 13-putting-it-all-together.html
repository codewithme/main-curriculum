So how should we think about it?

If at any point, you're not sure what to do next, ask yourself these questions:

  1. Do I have my content?
  2. Have I structured my content with HTML?
  3. Have I styled my content with CSS?
  4. Do I want to add some user interaction with jQuery?
  5. Does it all work?
